# Cybersecurity & Automation Learning Roadmap (30 Days)

## Objective
This roadmap focuses on building practical cybersecurity skills for entry-level roles (e.g., SOC Analyst / Security Analyst), while incorporating automation and low-code capabilities.

---

# Phase 1: Fundamentals (Day 1–5)

## Goals
- Understand basic networking and web concepts
- Learn how web applications function
- Get familiar with essential security tools

## What to Do
- Study HTTP/HTTPS fundamentals
- Learn request/response structure
- Understand cookies and sessions
- Complete the following paths on TryHackMe:
  - Pre Security
  - Web Fundamentals
- Install and explore Burp Suite (intercepting requests, proxy basics)
- Explore Wireshark for basic packet inspection

## What You Will Learn
- How web communication works
- How data flows between client and server
- Basic usage of web security tools

---

# Phase 2: Hands-on Vulnerability Testing (Day 6–12)

## Goals
- Perform basic vulnerability testing
- Understand common web attacks
- Learn container-based environment setup

## What to Do
- Deploy DVWA (Damn Vulnerable Web Application)
- Practice SQL Injection (priority)
- Practice XSS (optional)
- Use Burp Suite to intercept and modify requests
- Learn basic Docker commands
- Deploy DVWA using Docker

## What You Will Learn
- How common web vulnerabilities work
- How to test web applications using security tools
- How to set up isolated testing environments using Docker

---

# Phase 3: Automation & Scripting (Day 13–20)

## Goals
- Develop basic scripting skills
- Automate simple security testing tasks

## What to Do
- Learn basic Python (requests library, loops, functions)
- Build a simple script to:
  - Send HTTP requests
  - Test for potential SQL Injection patterns
- Document usage and results

## What You Will Learn
- How to automate repetitive security tasks
- How scripts can assist in vulnerability testing
- Basic programming logic applied to cybersecurity

---

# Phase 4: Job Preparation (Day 21–30)

## Goals
- Prepare for job applications
- Strengthen understanding of key concepts
- Build confidence in explaining technical work

## What to Do
- Apply for entry-level cybersecurity roles
- Review and explain:
  - SQL Injection concepts
  - Burp Suite usage
  - Your testing workflow
- Learn basic SOC concepts:
  - What logs are
  - How to identify suspicious behavior

## What You Will Learn
- How to communicate technical knowledge
- How cybersecurity work is applied in real-world roles
- Basic understanding of security monitoring and analysis

---

# Expected Outcome

By completing this roadmap, you will have:
- Foundational knowledge of web security
- Hands-on experience with common tools
- Experience in vulnerability testing
- Basic scripting and automation capability
- A structured understanding of cybersecurity workflows

---

# Key Principle

Focus on practical execution and consistent output rather than passive learning.
